This article describes the formula syntax and usage of the IMREAL function in Microsoft Excel.

Description

Returns the real coefficient of a complex number in x + yi or x + yj text format.

Syntax

IMREAL(inumber)

The IMREAL function syntax has the following arguments:

  • Inumber Required. A complex number for which you want the real coefficient.

Remarks

  • Use COMPLEX to convert real and imaginary coefficients into a complex number.

Example
