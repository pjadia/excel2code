This article describes the formula syntax and usage of the IMSIN function in Microsoft Excel.

Description

Returns the sine of a complex number in x + yi or x + yj text format.

Syntax

IMSIN(inumber)

The IMSIN function syntax has the following arguments:

  • Inumber Required. A complex number for which you want the sine.
