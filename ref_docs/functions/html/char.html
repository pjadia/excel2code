This article describes the formula syntax and usage of the CHAR function in Microsoft Excel.

Description

Returns the character specified by a number. Use CHAR to translate code page numbers you might get from files on other types of computers into characters.

Operating environment

Character set

Macintosh

Macintosh character set

Windows

ANSI

The ANSI character set was the standard set of characters used in Windows operating systems through Windows 95 and Windows NT, after which Unicode was adopted. ANSI consists of 218 characters, many of which share the same numerical codes as in the ASCII/Unicode formats. https://www.techopedia.com/definition/932/ansi-character-set

Syntax

CHAR(number)

The CHAR function syntax has the following arguments:

  • Number Required. A number between 1 and 255 specifying which character you want. The character is from the character set used by your computer.

    Note: Excel for the web supports only CHAR(9), CHAR(10), CHAR(13), and CHAR(32) and above.
