This article describes the formula syntax and usage of the IMCONJUGATE function in Microsoft Excel.

Description

Returns the complex conjugate of a complex number in x + yi or x + yj text format.

Syntax

IMCONJUGATE(inumber)

The IMCONJUGATE function syntax has the following arguments:

  • Inumber Required. A complex number for which you want the conjugate.
