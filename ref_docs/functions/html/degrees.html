This article describes the formula syntax and usage of the DEGREES function in Microsoft Excel.

Description

Converts radians into degrees.

Syntax

DEGREES(angle)

The DEGREES function syntax has the following arguments:

  • Angle Required. The angle in radians that you want to convert.

Example

Copy the example data in the following table, and paste it in cell A1 of a new Excel worksheet. For formulas to show results, select them, press F2, and then press Enter. If you need to, you can adjust the column widths to see all the data.
